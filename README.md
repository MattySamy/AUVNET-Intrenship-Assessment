## Installation

Here it's the way to start this project

#### 1) Clone Project

```bash
git clone https://github.com/MattySamy/AUVNET-Intrenship-Assessment.git
```

#### 2) Navigate to the project directory:

```bash
  cd AUVNET-Intrenship-Assessment
```

#### 3) Install Dependencies:

```bash
npm install
```
